# Changelog

All notable changes to this repository will be documented in this file.

The format is based on Keep a Changelog, and this project uses Semantic Versioning for the portable skill package surface.

## [0.1.0] - 2026-03-17

### Added

- Initial public release of the `generate-demo-doc` skill package.
- Tool-agnostic skill manifest in `generate-demo-doc/skill.yaml`.
- Shared prompt instructions in `generate-demo-doc/PROMPT.md`.
- Codex, Claude Code, and Cursor adapter files.
- Demo document reference template in `generate-demo-doc/references/demo-template.md`.
- Contributor guide and example requests/output.

### Changed

- Reframed the repository from a Codex-only skill to a portable agent skill package with thin platform adapters.
